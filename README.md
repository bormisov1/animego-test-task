# Animego Test Task

## Docker

#### Run postgres db locally via docker-compose

```bash
docker-compose up
```

#### Stop postgres db via docker-compose

```bash
docker-compose down
```

#### Stop & Remove postgres db via docker-compose

```bash
docker-compose down --rmi all -v
```

#### Build

```bash
docker build -t animego-test-task .
```

## pnpm , npm

#### Installation

```bash
$ pnpm install
```

#### Running the app

```bash
$ npm run dev
```
